## Array Chunk Equals

#### Usage

```js
import chunkEquals from "array-chunk-equals";

const arr = [1, 2, 3, 4, 5, 6, 7];

console.log(chunkEquals(arr, 3));
// [[1, 2, 7], [3, 4], [5, 6]]
```

#### Install

with NPM

```sh
npm i array-chunk-equals
```

with Yarn

```sh
yarn add array-chunk-equals
```
